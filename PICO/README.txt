starters: this folder contains all basic level experimental code. Most, target specific issues in an enclosed, individual environment.

piCAN_send81: this folder contains every version of code only made sure to send messages to the CAN bus. No instance reads or
processes the external messages.

piCAN_watchdog: this folder contains multiple programs that read and process messages in the CAN bus, managing the information in different ways.

piCAN_system: this folder contains integral code, grouping the information from the other folders, to be applied in complete systems responding to various tasks.
